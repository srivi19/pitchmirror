# PitchMirror

PitchMirror is a web application that helps you practice your startup pitch and get feedback from AI investors. You can record your pitch, and the application will analyze it and provide you with a detailed report. You can also simulate a Q&A session with different AI investor personas to practice your responses under pressure.

## Features

- **Pitch Recording**: Record your pitch using your microphone or by typing it in.
- **Pitch Analysis**: Get a detailed analysis of your pitch, including scores for structure, storytelling, and clarity.
- **Investor Simulation**: Simulate a Q&A session with different AI investor personas.
- **Final Report**: Get a final report on your Q&A performance, including scores for clarity under pressure and confidence.

## Running the Application

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Run the development server**:
    ```bash
    npm run dev
    ```
3.  **Open the application**:
    Open your browser and navigate to `http://localhost:5173/` (or the port specified in the console).
